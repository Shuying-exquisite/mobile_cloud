# Mobile Cloud (移动云手机部落冲突专属版)

## 简介
移动云手机部落冲突专属版，防止应用休眠需要挂后台，或者挂服务器。

## 功能特性
- 防止云手机休眠
- 可挂后台运行
- 可挂服务器运行

## 安装与使用
1. 下载仓库中的可执行文件 `移动云手机部落冲突专属版.exe`。
2. 运行可执行文件，按照提示完成安装。
3. 按需选择挂后台或者挂服务器运行。

## 系统要求
- Windows操作系统
- 网络连接
